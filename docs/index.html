RBXTools

Open-source Roblox account management tools

Download Windows EXE View Source

Runs locally on your PC. No data collection. Not affiliated with Roblox.

Current Tool: Unfollow Manager

  • Unfollows all users you are currently following
  • Runs locally on your machine
  • Logs unfollows to a local file
  • Rate-limit handling built in

Security Warning

RBXTools uses your Roblox session cookie (.ROBLOSECURITY) to authenticate requests. Never share it or paste it into random websites.

  • ✅ Tool runs locally only
  • ❌ We do not collect or store cookies
  • ✅ Log out of Roblox after use to invalidate the cookie

How to use

  1. Download the EXE from Releases
  2. Run it and follow the prompts
  3. A local log file will be generated
  4. Log out of Roblox afterwards
