RBXTools

Open-source Roblox account management tools

What is RBXTools?

RBXTools is a collection of local, open-source utilities designed to help Roblox users manage their accounts responsibly.

All tools run locally on your computer. No data is collected, stored, or transmitted.

Current Tool: Unfollow Manager

  • Unfollow users you are following
  • Option to unfollow non-friends only
  • Optional filter for offline users
  • Automatic rate-limit handling
  • Generates a local log of unfollowed users

Security Warning

This tool uses your Roblox session cookie (.ROBLOSECURITY) to authenticate requests.

  • ❌ Never share your cookie
  • ❌ Never paste it into random websites
  • ✅ This tool runs locally only

After use, log out of Roblox to invalidate the cookie.
